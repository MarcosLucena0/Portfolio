# Portfólio Pessoal – Marcos Lucena


## ✨ Descrição
Este é meu portfólio pessoal desenvolvido para apresentar minhas habilidades, projetos e experiência como desenvolvedor full-stack freelancer. O site é uma vitrine profissional que permite aos visitantes conhecer melhor meu trabalho, formação e entrar em contato comigo para parcerias ou oportunidades.

## 📌 Funcionalidades
- Home: Apresentação pessoal com animações suaves juntamente com o menu com todas as sessões da página.

- Sobre: Breve resumo da minha trajetória, valores e foco profissional juntamente com tecnologias e ferramentas que domino.

- Projetos: Galeria interativa com os principais projetos que desenvolvi, com links para os repositórios ou deploys.

- Formação: Certificados e diplomas de cursos consluídos.

- Contato: Sessão apresentando meu e-mail, LinkedIn, GitHub e outras fomas de me encontrar.

- Responsivo: Design adaptado para desktop, tablet e mobile.

## 🛠️ Tecnologias Utilizadas
- Frontend: HTML5, CSS3, JavaScript.

- Animações: CSS Transitions.

- Hospedagem: GitHub Pages.
